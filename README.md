# Hotel Booking Database Project

## Domain Description
This project represents a hotel booking system where users can search hotels, view rooms, and make bookings.

## Entities
- Users
- Hotels
- Rooms
- Bookings

## Relationships
- A hotel has many rooms
- A room belongs to one hotel
- A booking belongs to one user and one room

## Features
- Store users, hotels, rooms, and bookings
- Track check-in and check-out dates
- Analyze bookings using SQL queries
